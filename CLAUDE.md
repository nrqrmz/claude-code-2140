## Project Overview

A Pokedex web application displaying the first 150 Pokemon, built with vanilla HTML, CSS, and JavaScript. Data is fetched from the [PokeAPI](https://pokeapi.co/).

## Development

Open `index.html` directly in a browser or use a local server (e.g., `npx serve` or VS Code Live Server extension).

## Structure

- `index.html` - Main HTML page
- `style.css` - Styles
- `main.js` - JavaScript
